# calculo_sueldo_neto
álculo de Sueldo Neto en Python

Este proyecto es un programa en Python que calcula el sueldo neto de un empleado.

El programa solicita al usuario:
- Sueldo bruto
- Otros descuentos

Luego calcula automáticamente:
- Descuento de Seguridad Social (TSS)
- Retención del ISR
- Bonificación
- Sueldo neto final

## Cómo usar el programa

1. Ejecutar el archivo `Sueldo_neto.py`
2. Ingresar el sueldo bruto
3. Ingresar otros descuentos
4. El programa mostrará todos los cálculos en pantalla.
